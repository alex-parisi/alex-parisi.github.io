Style Transfer Web App

This web app implements the "TF Hub Style Transfer Wrapper" package I created. It allows anyone to uploads two images - a content image and a style image - and then generate a new image that is the original content image with the transfered style of the style image.

This is an example of a typical input content and style image, with its associated style-transferred output image.

You can view the Github repo of the original package here: tfhub-styletransfer-wrapper

You can view the Github repo of the Flask web app here: styletransfer-webapp

The web app was written using Flask and Python 3.7, and hosted on Google Cloud's App Engine service. Please do not abuse this webapp as it is just for personal use.

Instructions:

Upload two images: one content image and one style image. The web app will regenerate the content image using the style of the style image. After selecting both the content and style images, press the Submit button below. Then, wait a little bit (may be as long as 30 seconds) for the web app to load and apply the TF Hub model. Once done, you should see the cropped content and style images, as well as the output stylized image.

If the content and style images are resized or cropped incorrectly, you can resize your content image to 512x512 and the style image to 256x256 before uploading to prevent the web app from removing any image content.

If you do not have any content or style images handy, you can use these examples: Content Image Style Image

Please note that this web app is very new and may still have some bugs, so please follow the instructions correctly and (if possible) let me know of any errors you encounter.
